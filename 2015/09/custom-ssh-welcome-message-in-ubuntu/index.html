自定义 ubuntu ssh 欢迎信息

作者 jk2K
发表于 2015-09-14 1 分钟阅读

步骤

  1. 打开/etc/update-motd.d, 这个目录下面的文件会在登录成功后按照序号的顺序先后运行
    1

    cd /etc/update-motd.d
  2. 我们找到00-header这个文件, 修改它, 在文件末尾添加如下代码
    1
2
3
4
5
6

    cat << "EOT"
  /\_/\
=( °w° )=
  )   (  //
 (__ __)//
EOT
  3. 重新登录服务器就会看到如下效果啦

小提示

重新登录服务器后也许不会看到效果, 你再登录一次就肯定会看到啦, 就酱
